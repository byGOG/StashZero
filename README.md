# 💎 StashZero: Studio Edition

**The Ultimate Neo-Glow Offline Installer & System Monitoring Hub.** 🚀✨📂

StashZero Studio Edition is a high-performance, aesthetically driven application distribution tool built with **Tauri**, **React**, and **Rust**. It transforms the mundane task of software installation into a premium, data-rich "Studio" experience.

## 🌟 Key Features

### 🏢 Professional Studio Layout
- **Sidebar-Centric Navigation**: Clean categorization of your installer library.
- **Neo-Glow Aesthetic**: Glassmorphism V3 with 24px background blur and emerald neon accents.
- **Dynamic Mesh Gradients**: A fluid, living background that responds to your presence.

### 📊 Extreme Telemetry Hub
- **Triple-Bar Monitoring**: Real-time tracking of **CPU**, **RAM**, and **Disk** usage.
- **Network Activity**: Live Download/Upload speeds (KB/s) with pulsing indicators.
- **Pro Diagnostics Panel**: Hover over the system badge to reveal:
  - **Hostname** & **OS Version**
  - **Kernel Build** & **Total Processes**
  - **CPU Model** & **Swap Memory**
  - **Local IP** & **Live System Uptime**

### ⚡ Performance-First Backend
- **Asynchronous Pipeline**: Built with `tokio`, ensures the UI remains 100% responsive even during massive silent installations.
- **Smart Silent Installs**: Automated `/S`, `/VERYSILENT`, and `/qn` switches for `.exe` and `.msi` files.
- **Post-Install Scripting**: Automatically detects and runs `post-install.ps1` for advanced configurations.
- **Icon Intelligence**: High-speed Rust-based icon extraction with local caching for instant loading.
